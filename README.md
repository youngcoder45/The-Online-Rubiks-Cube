# The Cube — 3D Interactive Cube

A small interactive 3D cube demo built with HTML, JavaScript and styles (Sass). The project renders a rotating cube effect and includes subtle UI/visual polish.

## Preview

- ![Cube preview 1](src/1.png)
- ![Cube preview 2](src/2.png)

> If the images don't show when viewing on GitHub, place `1.png` and `2.png` into the `src/` folder (next to `index.html`).

## Files

- `src/index.html` — Demo HTML that loads the cube and required scripts/styles.
- `src/script.js` — JavaScript powering the cube animation and interactions.
- `src/style.sass` — Styles written in Sass; compile to CSS if you prefer a single `.css` file.

There is also a `dist/` folder (if present) that may contain built/compiled output.

## Usage

1. Open `src/index.html` in a modern browser to view the demo.
2. If `style.sass` needs compiling, either use a Sass compiler or replace it with the compiled CSS in `dist/` if available.

## Tech Stack

- HTML5
- JavaScript (vanilla)
- Sass (style.sass)
